- 링크 스타일은 기존 의도대로 primary color(주황 계열) + underline 유지했습니다.
- 기존 TOC 스크롤 기반 하이라이트 구조는 그대로 유지했습니다.

## Next Action Item
- 실제 글(회고/컨퍼런스 카테고리)에서 아래 케이스를 추가 검증
  - 일반 bold 문구
  - bold를 포함한 링크 문구
  - inline code + bold 혼합 문장
- 필요 시 italic(`*text*`) 및 취소선(`~~text~~`)도 동일 파서에 확장

## 주의사항
- 현재 파서는 경량 커스텀 파서이므로, 완전한 CommonMark/GFM 스펙 전체를 지원하지는 않습니다.
- bold 정규식은 기본적인 `**text**` 패턴 중심으로 처리합니다(복잡한 중첩 케이스는 제한적).
