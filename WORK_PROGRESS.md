# 작업 진행상황 (게시글 상세 UI 복구)

## 진행상황
- 사용자 이슈("게시글 안 UI가 깨졌고, 글 제목 연동 사이드바가 사라짐")를 기준으로 상세 페이지를 복구했습니다.
- 상세 본문 렌더링 복구
  - 기존 `pre` 단일 출력 방식에서, 마크다운 구조를 블록 단위(제목/문단/리스트/코드)로 파싱해 렌더링하도록 변경
  - 결과적으로 본문이 한 덩어리 코드처럼 보이던 문제를 완화
- 제목 연동 사이드바(목차) 복구
  - 본문의 `##`, `###` 제목을 추출해 우측 고정(sticky) 사이드바 목차로 표시
  - 사이드바 항목 클릭 시 해당 본문 제목 anchor로 이동
- 기존 상세 헤더(카테고리/제목/날짜/설명/태그)와 목록 복귀 링크는 유지

## Next Action Item
- 인라인 마크다운(링크/볼드/코드 스팬)까지 파싱 정확도를 높이는 2차 개선
- 모바일에서도 접기 가능한 목차 UI 제공 여부 검토
- 긴 코드블록에서 언어 하이라이팅이 필요하면 라이브러리 기반으로 확장

## 주의사항
- 현재 파서는 경량 파서이므로 복잡한 마크다운 문법(중첩 리스트, 테이블, 고급 링크 문법)은 완전 대응하지 않습니다.
- 그래도 기존 "본문 전체가 깨져 보이는" 문제와 "사이드바 부재"는 우선 복구했습니다.

## 사용자가 해야할 일
- 변경 확인
  1. `pnpm install`
  2. `pnpm build`
  3. `pnpm dev`
  4. 게시글 상세 페이지에서
     - 본문이 문단/제목/리스트로 정상 보이는지
     - 우측 목차(글 제목)가 보이고 클릭 이동되는지 확인
