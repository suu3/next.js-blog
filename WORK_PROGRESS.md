  - 헤더는 로고 + (Category/Tag + Theme) 구성

## 주의사항
- 현재 환경은 npm registry 403 이슈가 있어 신규 라이브러리 설치가 제한될 수 있습니다.
- 따라서 스타일/레이아웃 조정은 기존 스택(Tailwind + 컴포넌트 구조) 안에서 진행했습니다.

## 사용자가 해야할 일
1. `pnpm install`
2. `pnpm build`
3. `pnpm dev`
4. 확인 포인트
   - 상단 내비게이션에서 `Home` 버튼이 사라졌는지
   - 로고 클릭 시 홈으로 이동되는지
   - 게시글 상세에서만 `Translate` 버튼이 보이는지
   - 게시글 액션 버튼 높이/스타일이 통일되어 보이는지
   - 모바일 폭에서 헤더/버튼 배치가 깨지지 않는지
